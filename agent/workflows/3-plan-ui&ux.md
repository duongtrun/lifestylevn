Khi tôi yêu cầu lên kế hoạch UI/UX, bạn phải tư duy như một Product Designer & UX Expert trước khi làm một Coder. 

**Bắt buộc trước khi làm:**
1. **Tham khảo & Phân tích:** Đưa ra ví dụ hoặc góc nhìn từ các ứng dụng SaaS lớn (như Linear, Vercel, Stripe, Notion...) để làm chuẩn mực cho thiết kế chuyên nghiệp, hiện đại.
2. **Tư duy tổng thể (Holistic View):** Đánh giá xem UI hiện tại có bị rối, chật chội hay thừa thao tác không? Đề xuất các cơ chế thông minh (ví dụ: Collapse/Expand, Smart Toggle, Hover Effects, Glassmorphism) để tối ưu không gian và giảm thiểu số lần click của người dùng.
3. **Chiến lược Responsive:** Phải trình bày rõ giao diện sẽ co giãn như thế nào trên 3 màn hình: 
   - Desktop (Máy tính): Cấu trúc Grid, Sidebar.
   - Tablet (iPad): Thu gọn cột, điều chỉnh cỡ chữ.
   - Mobile (Điện thoại): Stack dọc, vuốt ngang, ẩn bớt chi tiết thừa.
4. **Hệ thống Typography & Spacing:** Định hình sẵn các cấp bậc cỡ chữ (Heading, Body, Caption) và hệ thống khoảng cách (Padding/Margin) để đồng bộ toàn platform, không code tùy hứng.

**Format Báo cáo Kế Hoạch UI/UX:**
1. **Phân tích vấn đề & Mục tiêu UX** 
   - *Giao diện cũ gặp lỗi gì? (ví dụ: bóp nghẹt diện tích, thao tác cồng kềnh, quá nhiều Select option)*
2. **Cảm hứng & Chuẩn mực thiết kế** 
   - *Học hỏi từ các app lớn nào? Phong cách thiết kế ưu tiên là gì? (Minimalist, Premium...)*
3. **Đề xuất Giải pháp UI thông minh** 
   - *Áp dụng cơ chế hiển thị nào? (ví dụ: Card Layout thay vì List, Inline Action thay cho Modal ngắt quãng).*
4. **Mockup Cấu trúc & Responsive**
   - *Vẽ sơ đồ khối bố cục (Sử dụng Mermaid hoặc text flow).*
   - *Trình bày rõ phương án cho Desktop / Tablet / Mobile.*
5. **Định chuẩn Style Guide**
   - *Cỡ chữ, màu sắc chủ đạo, bóng đổ (Shadows), bo góc (Radius).*
6. **Chia Phase triển khai Code UI**
   - *Ví dụ: Phase 1 tạo UI Component góc -> Phase 2 lắp vào trang chủ -> Phase 3 bổ sung Animation.*

> **Lưu ý:** Tuyệt đối không nhảy vào code ngay! Lên kế hoạch /plan-ui&ux xong phải chờ tôi duyệt "OK" rồi mới được dùng lệnh code để thực thi.
